Швидкий старт

1. Встановлення NovaChain CLI

Установіть офіційний CLI інструмент для роботи з контрактами та мережею.

npm install -g novachain-cli

2. Ініціалізація проєкту

Створіть новий стандартний шаблон проєкту для швидкого старту.

novachain init my-dapp

3. Деплой контракту

Після компіляції відправте контракт у testnet.

novachain deploy --network testnet

FAQ для розробників

Які мови підтримує NovaChain?

Основна мова смарт-контрактів — TypeScript-based синтаксис, адаптований для виконання у VM NovaChain.

Як деплоїти смарт-контракт?

За допомогою CLI: novachain deploy. Контракт компілюється у байткод і надсилається в мережу.

Чи є тестова мережа?

Так, існує testnet, доступний через RPC: https://rpc.testnet.novachain.app.

Яка модель оплати газу?

NovaChain використовує адаптивний gas-mechanism, залежно від складності операції та навантаження мережі.
