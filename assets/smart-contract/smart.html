Smart Contracts & Tools

Як писати, тестувати та деплоїти смарт-контракти у мережі NovaChain.

Почати SDK DevTools

Швидкий старт

1. Встанови NovaChain CLI

npm install -g novachain-cli

2. Створи проект

novachain init my-dapp

3. Деплой контракт

novachain deploy MyContract.sol

SDK

Легке підключення до NovaChain через JavaScript / TypeScript.

npm install @novachain/sdk

// Підключення до RPC
import { Nova } from "@novachain/sdk";

const client = new Nova("https://rpc.testnet.novachain.app");

// Читання балансу
const bal = await client.getBalance("0x123...");
console.log(bal);

Testnet Faucet

Отримай тестові токени NVC для розробки.

Працює тільки для testnet NovaChain.

Приклади смарт-контрактів

Storage.sol


pragma solidity ^0.8.0;

contract Storage {
    uint public value;

    function set(uint _v) public {
        value = _v;
    }
}

Token.sol


pragma solidity ^0.8.0;

contract Token {
    mapping(address => uint) public balance;

    function mint(address to, uint amount) public {
        balance[to] += amount;
    }
}

Dev Tools

Localnet

Запускай локальний тестовий вузол NovaChain за секунди.

Contract Inspector

Переглядай структуру контрактів, байткод та сигнатури функцій.

RPC Explorer

Тестуй RPC виклики та взаємодію з мережею прямо в браузері.

Gas Profiler

Аналізуй газ, оптимізуй контракти та зменшуй витрати.

FAQ

Чи сумісна NovaChain з Solidity?

Так, мережа повністю підтримує EVM.

Де брати тестові NVC?

Через Faucet або CLI командою novachain faucet.

Чи є локальна мережа?

Так, локальний нод запускається через novachain localnet.
